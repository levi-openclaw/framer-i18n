Add multi-language support to your Framer projects. Create, manage, and export translations for internationalization.

## What It Does

**Framer i18n** helps you:
- Manage multiple languages in one place
- Create translation keys for all your text
- Edit translations side-by-side
- Export ready-to-use JSON files

## How to Use

### 1. Install
1. Download `plugin.zip` from this repo
2. Go to [Framer Marketplace Dashboard](https://www.framer.com/marketplace/dashboard/plugins/)
3. Click "Add Plugin" ‚Üí Upload `plugin.zip`

### 2. Add Languages
1. Open the plugin in Framer
2. Click **+ Add** to add a new language
3. Choose from preset languages or add custom ones

### 3. Create Translation Keys
1. Click **Edit Translations**
2. Click **+ Add Key**
3. Enter a key name (e.g., `hero.title`, `nav.home`)

### 4. Add Translations
1. Select a language from the tabs
2. Enter translations for each key
3. See previews of other languages side-by-side

### 5. Export & Use
1. Click **Export JSON**
2. Add the JSON to your Framer project
3. Use the keys in your text layers or code
