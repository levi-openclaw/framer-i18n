- **Side-by-side editing** ‚Äî Edit one language, see others
- **JSON export** ‚Äî Ready to use in your project

## How to Install

1. Download `plugin.zip` from this repo
2. Go to [Framer Marketplace Dashboard](https://www.framer.com/marketplace/dashboard/plugins/)
3. Click "Add Plugin" ‚Üí Upload `plugin.zip`

## How to Use

### 1. Add Languages
Click **+ Add** to add a new language. Choose from presets (English, Spanish, French, German, etc.) or add custom ones.

### 2. Create Translation Keys
Click **Edit Translations** ‚Üí **+ Add Key**. Use dot notation: `hero.title`, `nav.home`, etc.

### 3. Add Translations
Select a language tab, enter translations for each key.

### 4. Export & Use
Click **Export JSON** and add to your Framer project.
